Weather Dashboard

Your go-to app for accurate and up-to-date weather forecasts

Key Features

Real-Time Weather Updates

Provides up-to-the-minute weather information for your chosen location, ensuring you have the most accurate and current weather data at your fingertips.

Interactive Search Functionality

Allows users to search for weather information by city name, providing a quick and easy way to find the weather forecast for any location worldwide.

Detailed Weather Insights

Offers comprehensive weather details including real feel temperature, humidity levels, wind speed, and atmospheric pressure, giving users a complete understanding of current weather conditions.

About the Project

The inspiration behind this weather dashboard project comes from a personal need to have a reliable source of weather information that is both accurate and easy to use. This project is part of the Portfolio Project for Holberton School.
